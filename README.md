# JE_EJ
英語できないからCLIツールを作ってみる試み

## 使用にあたって

以下のサイトに従ってGASで関数を作成してください

https://qiita.com/tanabee/items/c79c5c28ba0537112922

##　使い方

.envファイルを実行ファイルと同じ場所に作成し

```
URL=<作成したURL>/exec?
```
を記述

### 翻訳する場合

    `jeej <翻訳する単語の言語> <翻訳後単語> <翻訳する単語>`
```
    例
        jeej ja en こんにちは

    出力例
        Hello
```

### 対応言語の確認

    `jeej -c`

### バージョンのチェック

    `jeej -v`

## しょうもない点

英語　⇔　日本語しか対応してないです。
